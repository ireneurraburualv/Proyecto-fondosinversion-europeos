# Proyecto Fondos Inversion Europeos
El objetivo principal del proyecto es desarrollar un modelo útil que permita predecir las comisiones de gestión de los fondos de inversión en función del tipo de fondo y sus características. Para ello se construirán distintos modelos de aprendizaje automático y se compararán los resultados obtenidos con el fin de descubrir aquel que ofrece mejores resultados. Asimismo, se tienen una serie de objetivos específicos que servirán para desmenuzar la idea general e ir avanzando por la misma. Estos objetivos específicos siguiendo el ciclo de vida del dato son los siguientes: 

# Extracción
•	Realizar una recopilación de datos adecuada y en base a las necesidades de negocio.
•	Estructurar las variables según su significado y su importancia.

# Procesado
•	Hacer una limpieza de los datos; dividir la base de datos, eliminar filas y columnas con baja cobertura, crear nuevas columnas, realizar técnicas de imputación de datos, eliminar outliers.
•	Estudiar las correlaciones y hacer una selección de las variables a utilizar en los modelos.

# Modelo de datos
•	Seleccionar las variables para los modelos.
•	Segmentar el data set original en tres data sets diferentes, uno por cada tipo de fondo de inversión.
•	Dividir los datos en datos de entrenamiento y datos de prueba.

# Análisis
•	Seleccionar el modelo de aprendizaje automático adecuado.
•	Analizar los resultados obtenidos.
•	Representar gráficamente los resultados obtenidos.
